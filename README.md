# ALATPay SDK (Javascript/Typescript)

An ALATPay client SDK for the javascript runtime.

## Features

- Typescript support
- Implements methods matching all of ALATPay's public API.

## Installation

### npm

```bash
npm i @gray-adeyi/alatpay-sdk
```

### yarn

```bash
yarn add @gray-adeyi/alatpay-sdk
```

### pnpm

```bash
pnpm i @gray-adeyi/alatpay-sdk
```

### bun

```bash
bun add @gray-adeyi/alatpay-sdk
```

### deno

```bash
deno add @gray-adeyi/alatpay-sdk
```
